Next Steps

In order to run Pegasus workflows you will also need to install HTCondor. You can download HTCondor from the HTCondor website: http://cs.wisc.edu/htcondor

The disk image you downloaded contains a PDF version of the Pegasus User Guide. Chapter 2 of the user guide contains a tutorial that will familiarize you with the basic usage of Pegasus. More information about Pegasus is available on the Pegasus website: http://pegasus.isi.edu.

If you have problems, or need help creating and running workflows, please contact Pegasus support (pegasus-support@isi.edu) or the Pegasus users mailing list (pegasus-users@isi.edu).
